# Latimore Life Legacy (Next.js)

## Local run
```bash
npm install
npm run dev
```

## One-click deploy to Vercel
1) Create a GitHub repo and push this project.
2) In Vercel, click **New Project** → Import the repo → Deploy.

If you want a deploy button for your README, replace `YOUR_GITHUB_URL` below after you create the repo:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=YOUR_GITHUB_URL)
